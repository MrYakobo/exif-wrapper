# ugly wrapper around mattwilson1024/google-photos-exif

## How do I run it?

```bash
git clone --recurse-submodules https://github.com/MrYakobo/exif-wrapper.git
cd exif-wrapper

echo "TAKEOUT_DIR=/home/user/Downloads/Takeout" > .env
docker-compose up
```
